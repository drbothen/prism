### F-LP6-LOW-1 â€” Detailed Fix Narrative

The gap was that both `extract_source_names_recursive` (materialization.rs)
and `extract_sources_from_ast` (explain.rs) matched only
`SqlStatement::Select`, leaving `SqlStatement::Dml` to fall through the
wildcard arm â€” a latent capability-gate bypass that would activate once
S-3.07 Phase 3 wiring sends DML nodes through the Layer 1 sensor fan-out
path.

**materialization.rs fix (lines 882-894):**

Added `Ast::Sql(SqlStatement::Dml(dml))` arm immediately after the
`Ast::Sql(SqlStatement::Select(select))` arm. The new arm:
- Walks `dml.source_select` (an `Option<SqlQuery>`) via `walk_sql_query`
  to discover sensor sources in subquery positions
- Walks `dml.filter` (an `Option<Predicate>`) via `walk_predicate` to
  discover sensors referenced in DML WHERE clauses that include InSubquery
- Carries a lineage anchor comment at line 887 tracing back to
  F-LP3-CRIT-1 â†’ F-LP4-MED-1 â†’ F-LP5-LOW-1 â†’ F-LP6-LOW-1 for
  reviewer auditability

**explain.rs fix (lines 490-506 + helper 532-560):**

Added matching `Ast::Sql(SqlStatement::Dml(dml))` arm in
`extract_sources_from_ast`. Because explain.rs uses a different internal
pattern than materialization.rs (closure-based dedup rather than mutable
vec threading), required refactoring `push_dedup` from a closure to a
free function (`collect_predicate_sources_into` at lines 532-560) that
accepts `&mut sources` â€” enabling the DML filter arm to thread sources
correctly.

`collect_predicate_sources_into` handles `InSubquery`, `Logical`, and
`Not` predicate variants, which are the subquery-bearing arms that can
contain sensor `SourceRef`s. This is semantically equivalent to
materialization's `walk_predicate`.

**Doc propagation (materialization.rs:852-871, engine.rs:594-598):**

Both function-level doc (materialization.rs) and gate-level doc (engine.rs)
now explicitly enumerate DML `source_select` + `filter` coverage alongside
the previously-documented SELECT and Pipe modes. No stale "SELECT-only"
claims remain in security-critical walker documentation.
