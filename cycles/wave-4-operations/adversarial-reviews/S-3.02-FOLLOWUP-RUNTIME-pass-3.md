**Severity:** MEDIUM — Standing Rule 3 #1 (surface-and-defer masquerading as fix).

```rust

    let plan = classify_predicates(&eq_exprs, &[]);  // empty spec slice
    let _ = plan;  // ← discarded — values come from direct extraction below
    let mut filters = FilterMap::new();
    for expr in &eq_exprs {

            filters.insert(col, val);
        }
    }
    filters
}
```

Actual filter map built from eq_exprs directly via extract_eq_filter_from_expr — functionally identical to deleted collect_eq_filters. **Recommendation:** Either (a) remove dead classify_predicates call + update doc; OR (b) actually thread per-sensor ColumnSpec slices through and use plan.push_down to drive filter map (true wiring per ADR-022 §C).
