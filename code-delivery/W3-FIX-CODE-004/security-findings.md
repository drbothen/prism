## Findings

### LOW-001 (Informational only — confidence 0.75, below blocking threshold)

**File:** `crates/prism-dtu-armis/src/routes/devices.rs`
**Category:** auth_bypass (design trade-off)

The `DTU_DEFAULT_INSTANCE_ORG_ID` sentinel path intentionally allows requests
without `X-Org-Id` for backward compat on default-instance clones. This is a
deliberate design choice (EC-003 in story spec), not a regression introduced
by this PR. The bypass only applies when `instance_org_id == DTU_DEFAULT_INSTANCE_ORG_ID`,
a server-side constant set at construction time — not client-controllable.
Real-org clones enforce auth model A unconditionally.

**Not blocking.** Informational note for operational runbooks.
