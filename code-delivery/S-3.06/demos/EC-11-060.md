# EC-11-060 — Write Verb in Non-Terminal Pipe Position

**Story:** S-3.06 v1.7 | **BC:** BC-2.11.004 | **Status:** PASS

## Edge Case

Write verb in non-terminal pipe position (e.g. `| contain | where ...`) returns
`E-QUERY-024`; no AST produced.

## Test Names

- Integration: `test_ac2_write_stage_not_terminal`
- Integration: `test_BC_2_11_004_two_write_verbs_in_sequence_rejected`
- Unit (pub-crate): `test_BC_2_11_004_parse_pipe_with_write_verb_not_terminal`
- Error constructor: `test_BC_2_11_004_error_024_message_contains_code_verb_and_position`
