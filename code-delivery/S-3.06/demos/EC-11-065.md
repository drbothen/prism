# EC-11-065 — WriteVerbRegistry Initialized with Empty Verb Set

**Story:** S-3.06 v1.7 | **BC:** BC-2.11.004 | **Status:** PASS

## Edge Case

`WriteVerbRegistry` initialized with empty verb set (no sensor specs loaded) causes any
terminal identifier in pipe position to produce `E-QUERY-023`.

## Test Names

- Integration: `test_ec11_065_empty_registry_unknown_verb`
- Integration: `test_BC_2_11_004_filter_mode_empty_registry_no_panic`
- Unit (pub-crate): `test_BC_2_11_004_parse_pipe_with_write_empty_registry_any_verb_023`
- Unit (pub-crate): `test_BC_2_11_004_reject_write_verbs_in_filter_empty_registry_always_ok`
- Unit (pub-crate): `test_BC_2_11_004_registry_is_empty_populated_vs_default`

## Evidence

### Integration — no panic (public API)

```
test test_ec11_065_empty_registry_unknown_verb ... ok
```

Query: `FROM crowdstrike_hosts | someidentifier` — public API, no panic.

### Unit — empty registry produces E-QUERY-023

```
test test_BC_2_11_004_parse_pipe_with_write_empty_registry_any_verb_023 ... ok
```

Registry: `WriteVerbRegistry::default()` (empty)
Query: `FROM crowdstrike_hosts | someidentifier`

Assertions:
- `result.is_err()`
- `result.unwrap_err()[0].message.contains("E-QUERY-023")`

Error message with empty registry:
```
E-QUERY-023: unknown write verb 'someidentifier' — no write verbs are registered for this sensor
```

### Unit — empty registry and filter mode

```
test test_BC_2_11_004_reject_write_verbs_in_filter_empty_registry_always_ok ... ok
```

Per `INV-FILTER-EMPTY-REGISTRY`: with an empty registry, `reject_write_verbs_in_filter`
always returns `Ok(())` — no verbs are known, so nothing can be rejected.

### Unit — is_empty

```
test test_BC_2_11_004_registry_is_empty_populated_vs_default ... ok
```

```rust
let empty = WriteVerbRegistry::default();
assert!(empty.is_empty());

let populated = test_registry(&["contain"]);
assert!(!populated.is_empty());
```

## Result

PASS — Empty registry produces E-QUERY-023 for any terminal pipe identifier; filter mode
accepts all inputs when registry is empty (no false-positives).
